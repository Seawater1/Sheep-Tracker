How It Works

  1. Tracker sends GPS and battery data to TTN over LoRaWAN.
  2. TTN uplink formatter turns the binary payload into fields.
  3. TTN webhook posts each uplink to the Sheep Tracker server.
  4. The dashboard polls that server and shows your dog or sheep on the map.
